This project is a mini e-commerce module built as part of the **Naksh Jewels ReactJS & Node.js Internship Assessment**.  
It demonstrates frontend development with React, backend APIs with Node.js & Express, and containerization using Docker.

## Tech Stack

### Frontend
- React (Vite)
- Context API for state management
- Functional components only
- Plain CSS (no UI libraries)

### Backend
- Node.js
- Express.js

### DevOps
- Docker
- Docker Compose


## Features

### Frontend SETUP
cd frontend
npm install
npm run dev

Frontend will run at:
http://localhost:5173

### Backend SETUP
cd backend
npm install
npm start

Backend will run at:
http://localhost:5000


### DOCKER SETUP
Docker & Docker Compose are required
Needed:
Docker Desktop installed
Docker Desktop running
WSL2 enabled (Windows users)

### Run using Docker Compose
rom the project root directory: docker compose up --build

### ACCESS THE APPLICATON
Frontend:
http://localhost:3000

Backend API:
http://localhost:5000/products
